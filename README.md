# Marvel API
Это веб-приложение, которое получает данные о героях и комиксах Marvel через API с официального портала и создает пользовательский интерфейс. 
### Stack:
+ Фреймворк: React
+ Стили: SCSS
+ Роутинг: React Router
+ API: Marvel API
